# Dirty Tricks

Dirty Tricks automatically creates and updates redirect macros for Rogues and Hunters so your threat-transfer abilities always point to the right target with minimal setup.

## What It Does

- Creates and maintains redirect macros for:
  - Tricks of the Trade (Rogue)
  - Misdirection (Hunter)
- Detects tanks in party and raid groups and updates targets automatically.
- Supports solo Hunter play by using your pet as a redirect target when appropriate.
- Allows forcing a specific target through settings, quick-select buttons, target selection, or slash commands.
- Provides a minimap icon for quick access to settings.

## Slash Commands

- `/dirtytricks` - Toggle the settings window.
- `/dirtytricks help` - Show command help.
- `/dirtytricks toggle` - Enable or disable the addon behavior.
- `/dirtytricks settank <name>` - Force a specific tank name.
- `/dirtytricks cleartank` - Clear forced tank and return to auto-detection.
- `/dirtytricks minimap` - Toggle minimap icon visibility.
- `/dirtytricks debug` - Print current detection and macro debug details.
